How to Install Linux Mint (With Pictures)

  • Nahashon Muiru
  • Aug 10, 2023

Linux is an open source operating system that comes in many different variants. What sets apart these is mainly their package manager e.g, APT, PACMAN, RPM etc. but most the high level functionality is basically the same for all. Under these distributions are hundreds of linux distros from Manjaro, Linux mint, Kali Linux, Parrot, Gentoo, Tails, Android, Red Hat, Fedora etc. The choice on which distro to use is usually a deduction based on the need or simply just preference. For instance security researchers and pentesters would go for kali or parrot in most cases while basics users or learners seeking windows alternatives on linux would go for Ubuntu.

In this article we'll be covering how to install Linux Mint to disk/drive.

1

Assuming you already have a linux mint bootable USB stick or CD or DVD and you've booted up from it, you'll be greeted with the following screen. This is a live environment, a unique feature with linux that allows users run the OS directly from the installation media without installing. However any changes made are lost on reboot unless one has set up persistence. The next thing you need to do is to double click the disk icon labeled Install Linux Mint. Note: If you dont know how to make USB drive bootable or boot from it, check out this article

2

On the new pop-up screen, you'll then be prompted to choose your language as shown below and when you're done click continue.

3

The next step involves selecting your keyboard layout preference from the list. A testing space is also provided for you to test your keys and affirm that indeed that is how you want your keyboard to function when certain key combinations are executed.

4

To have the most recent updates when installing the system, connect to your wireless network as shown below.

5

Optionally you can install multimedia codes.

6

At this point we're now going to select the installation type from the three available choices.

  • You can choose to install alongside an already existing operating system (for this case you can see kali is already installed). This will be a fairly easy process in partitioning and you'll be prompted on which operating system you want to boot into during system boot up via a bootloader.
  • Choice two allows you to completely erase your disk including all files and operating systems present inherently creating new space for the new linux installation. Note: Be cautious with this selection.
  • Choice three allows you to exercise more discretion and we will see how this works as this is the option we go with.

    • 7

    We should now be able to see the list of partions available. Select the partion you intend to install to being careful not to choose the wrong one. For this case i had already set up my 30GB partion labeled as /dev/sda4 so i'm sure its the one because of its size. Click change and a small window will pop up, you can resize it as you want, for this case i leave it as it is. Choose your filesystem on the second step on which i advice you go with Ext4 journaling file system and then tick format. On the third step choose root '/' as your mount point then click OK.

    8

    Once that is done you should have such a screen depending on your partition of choice, mine remember is labeled /dev/sda4, yours could be different. Proceed to click Install Now.

    9

    You will then be prompted to select your location from a map that is, more or less your domicile country's capital.

    10

    Now enter your name and your computer name should auto fill. The rest should be easy for you. Additionally, you can encrypt your home folder and its contents to avoid unauthorised access.

    11

    Now be patient as the system installs to disk.

    12

    Once that is done, restart your computer, and you will be greeted with such on log on.

    If you encountered any problems at any of the steps, be sure to contact me and i'll guide you through.
